# Multipass
Dicas Multipass dia a dia


<h1>Instalando</h1>
<h2>sudo snap install multipass</h2>


<h1>Criar</h1>
<h2>multipass launch -n k8s -c 2 -m 2G -d 20GB</h2>

<h1>Deletar</h1>
<h2>multipass delete<br>
multipass purge</h2>


<h1>Listar</h1>
<h2>multipass list


<h1>Conectar</h1>
<h2>multipass shell k8s<br>
multipass exec k8s --name -r
</h2>
